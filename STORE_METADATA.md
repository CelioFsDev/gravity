## 📝 NOTAS IMPORTANTES

1. **Política de Privacidade**: Deve ser uma página web pública HTTPS. Considere criar em https://catalogoja.app/privacy

2. **Termos de Serviço**: App Store exige, Play Store recomenda. Crie em https://catalogoja.app/terms

3. **Conta de Exclusão**: Ambas exigem funcionalidade de exclusão de conta. Você já tem a tela `/legal/delete-account`

4. **Screenshots**: Use device frames para melhor apresentação (Figma, AppMockUp, etc)

5. **Versioning**: Sempre aumente `versionCode` antes de submeter atualizações
   - Android: `versionCode` é número inteiro (1, 2, 3...)
   - iOS: versão semântica (1.0.0, 1.0.1, 1.1.0...)

6. **Build & Release**:
   - Android: Gerar AAB (Android App Bundle) para Play Store
   - iOS: Enviar via Xcode ou fastlane para App Store
